
# Selection Sort Implementation

## Best Case
### It occurs when the array is already sorted
#### O(n^2)

## Worst Case 
### The array is in descending order
#### O(n^2)

## Average Case
### It occurs when the elements of the array are in jumbled order
#### O(n^2)
